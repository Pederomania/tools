# Paquetes Instalados

Este script lista todas las librerías instaladas en el entorno Python actual y muestra su versión.

## Uso

1. Clonar o descargar el repositorio.
2. Situarse en la carpeta del script:
```bash
cd paquetes_instalados
```

## Ejecutar:
```
python paquetes_instalados.py
```

## Ejemplo de salida:
```
numpy==1.26.4
torch==2.1.2
pandas==2.1.1
matplotlib==3.8.1
...
```

## Compatibilidad:
```
Python 3.8+
Funciona en cualquier sistema operativo (Windows, Linux, MacOS)
```
